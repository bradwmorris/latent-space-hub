# Sign Up for Sessions

- Human: In Discord, run `/paper-club` or `/builders-club` and follow prompts. No need to `/join` first — if you don't have a profile yet, one is created automatically from your Discord info.
- Agent helping human: Help choose a date and format the response exactly as Slop asks. Note: `/join` is not required — profiles are auto-created on first scheduling.
- Reference: [Slop Bot slash commands](/docs/slop-bot#slash-commands).
