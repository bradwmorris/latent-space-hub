# Use /join

- Human: In Discord, run `/join` with Slop. This is optional — if you schedule a Paper Club or Builders Club session first, a basic profile is created automatically from your Discord info. `/join` lets you enrich your profile with interests, role, company, and more.
- Agent helping human: Confirm `/join` succeeded and explain it creates or enriches the member profile with additional details beyond the basics.
- Reference: [Slop Bot slash commands](/docs/slop-bot#slash-commands).
