# Start Here

Latent Space Hub stores podcasts, articles, ainews issues, and derived entities.

Workflow:
1. Use `ls_get_context`.
2. Use `ls_search_nodes` before creating new records.
3. Use `ls_search_content` for chunk-level retrieval.
4. Use `ls_get_nodes` to inspect full records.
5. Use `ls_add_node` and `ls_create_edge` when adding new knowledge.
